## 项目上下文

### 项目信息
- **项目名称**: koatty_container
- **版本**: 2.0.6
- **类型**: IoC 容器库
- **Node.js**: >= 20.0.0
- **TypeScript**: strict mode 已启用
- **编译目标**: ES2023

### 关键特性
1. **Lazy Proxy**: 已实现延迟代理机制用于解决循环依赖
   - 位置: `src/utils/lazy_proxy.ts`
   - 使用 Symbol 标记代理状态
   - 支持错误缓存和延迟解析

2. **TypeScript 严格模式**: 
   - strict: true
   - noImplicitAny: true
   - noUncheckedIndexedAccess: true
   - experimentalDecorators: true
   - emitDecoratorMetadata: true

3. **IoC 容器架构**:
   - 容器核心: `src/container/`
   - 装饰器: `src/decorator/`
   - 处理器: `src/processor/`
   - 管理器: `src/manager/`
   - 工具类: `src/utils/`
